Deployment Options

              Choose how you want this to launch on your Windows 11 server.
            

            

              

                
Option A: Setup Installer

                
Recommended. Build the Windows setup file, copy it to the server, install it, then open Dragonwilds Server Control from the desktop or Start Menu.

              

              

                
Option B: MSI Package

                
Use this if your server setup prefers MSI deployment. The MSI build command creates a Windows installer package from the same desktop app.

              

              

                
Option C: Service Later

                
Use a Windows service only if you later want remote access from another PC. The installed desktop app is simpler and safer for local server control.
