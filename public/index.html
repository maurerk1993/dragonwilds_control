Deployment Options

              Choose how you want this to launch on your Windows 11 server.
            

            

              

                
Option A: Desktop Launcher

                
Recommended for the first server install. Copy the app folder to the Windows server, install Node.js 20 or newer, then double-click the launcher.

              

              

                
Option B: Electron Desktop App

                
Best final polish. The same UI can be wrapped into an installer with a desktop icon and bundled Node runtime.

              

              

                
Option C: Windows Service + Web UI

                
Best if you want to control the server from another PC. The service runs in the background and exposes the dashboard on your LAN or behind VPN.
